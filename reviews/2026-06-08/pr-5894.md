## Verdict: APPROVE

## Summary
Introduces a `ResourceWatcher` child component in `EditButton.tsx` that keeps a live WebSocket watch on the Kubernetes resource while the editor is open, forwarding resourceVersion changes into the `EditorDialog`. The dialog now detects external modifications (resourceVersion differs + user has unsaved edits) and shows an inline `Alert` instead of silently overwriting editor content. A Vitest integration test covers the main conflict scenario end-to-end.
