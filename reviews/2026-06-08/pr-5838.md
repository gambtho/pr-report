## Summary
Fixes Age column text clipping in the ObjectEventList details panel by making `HoverInfoLabel` in `Label.tsx` properly merge caller-supplied `labelProps.sx` with its default `whiteSpace: nowrap` style using MUI v5's sx array syntax. `ObjectEventList` then passes `{ whiteSpace: 'normal', overflowWrap: 'anywhere' }` to allow the timestamp label to wrap in narrow panels. 67 of the 69 changed files are auto-updated storyshots (expected MUI class-hash regeneration from the sx change).

## Findings

### Positive
- MUI v5 sx array merging (`sx={[defaultSx, ...(Array.isArray(labelSx) ? labelSx : labelSx ? [labelSx] : [])]}`) is the correct idiomatic pattern; handles undefined, object, and array forms of the caller's sx prop.
- Using `overflowWrap: 'anywhere'` rather than `break-word` is appropriate here — timestamp strings like "90d" don't contain natural break points, and `anywhere` ensures they always wrap at the panel boundary.
- A third mock event with `source: { component: 'kubelet' }` was added to the `ObjectEventList.stories.tsx` fixture, which also validates the source-guard logic added in the related PR #5824.
