## Verdict: APPROVE

## Summary
Removes the `useMemo(() => getTablesRowsPerPage(rowsPerPageOptions[0]), [])` that locked the default rows-per-page at mount time, replacing it with a direct call on each render. This ensures `SimpleTable` picks up settings changes made by the user without requiring a page refresh.

## Findings

### Suggestions
- **[LOW]** `frontend/src/components/common/SimpleTable.tsx:178` — `getTablesRowsPerPage` now runs on every render. Since it reads from localStorage, consider verifying it is synchronous and cheap — if so, a brief comment confirms the intent and avoids a future premature-optimisation reversal.
