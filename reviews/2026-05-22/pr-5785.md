## Verdict: APPROVE

## Summary
Replaces the deprecated `addListener`/`removeListener` MediaQueryList API with the standard `addEventListener('change', ...)`/`removeEventListener('change', ...)` in `themes.ts` and updates the three corresponding test cases in `themes.test.ts`. The handler type is also narrowed to `MediaQueryListEvent` (removing the overloaded `| MediaQueryList` union), which is the correct type for the `change` event callback.

## Findings

### Positive
- Test cases now also assert the correct event name (`'change'`) and verify the exact handler function reference is passed to removeEventListener — stronger coverage than the prior tests.
