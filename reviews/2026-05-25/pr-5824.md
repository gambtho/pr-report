## Verdict: APPROVE

## Summary

Fixes a runtime crash in the Events table's "From" column when Kubernetes returns events without a `source` field (common for events from `events.k8s.io` v1 API). Changes `item.source.component` to `item.source?.component ?? ''`. Minimal, correct, well-described fix.

## Findings

### Positive
- `item.source?.component ?? ''` correctly handles both `source` being `undefined` and `component` being `undefined` within a populated `source` object, matching the Kubernetes API spec where `EventSource` and its fields are optional.
