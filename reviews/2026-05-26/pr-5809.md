## Summary

The updated branch extends the milli-bytes fix to support decimal values (e.g. `1.5m` → `0.0015`). The regex `/^\d+(?:\.\d+)?m$/` correctly handles both integer and decimal milli-byte strings; `parseFloat(value.slice(0, -1)) / 1000` is the right arithmetic. Test coverage is complete with real kubectl-observed values.
