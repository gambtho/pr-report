## Summary

This PR replaces the `tsc` type-check script with `tsgo` (the Go-native TypeScript compiler preview) and changes `moduleResolution` from `"node"` to `"bundler"`. The performance gain (18s → 3s) is significant, but the tooling is still labeled `dev.20260519.1` — a pre-release dev build pinned to a specific date. The branch also carries a `needs-rebase` label. No human approval has been given by @illume. The `moduleResolution` change may affect plugin build semantics and warrants explicit maintainer sign-off before merge.
